# oh-my-lazybones

The marketplace for Agent Skills — discover, install, and share.

## Install

```bash
pip install oh-my-lazybones
```

## Quick Start

```bash
# Search for a skill
lazy search "email digest"

# Install a skill
lazy install daily-email-digest

# Use a remote API
LAZY_API_URL=https://api.lazybone.club lazy search "hello"
```
